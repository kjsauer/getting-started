## Launch Prompt
Windows: Start -> run -> cmd  
Git: Start -> run -> Git Bash  
Anaconda: Start -> run -> Anaconda (use Anaconda 2.7 to run Python 2.7)  

## Syntax
$ dir  
$ cd insertfoldername  
$ cd insertfoldername  
"ls" means list  
"ls bin"  
"." means current directory  
"pwd" means present working directory  

## Helpful Hints
Use tab completion!
